Instructions
Navigate the arrow to the star to win. Each command should be entered on a new line as shown in the example below. Note that each instruction can only be used twice and you cannot have two consecutive repeats.
move left (rotates arrow left 90 degrees and then move forward one step)
move right (rotates arrow right 90 degrees and then move forward one step)
move forward (arrow will move forward one step)
jump left (rotates arrow left 90 degrees and then move forward two steps)
jump right (rotates arrow right 90 degrees and then move forward two steps)
jump forward (arrow will move forward two steps)
repeat (repeats previous commands in order x 3)
